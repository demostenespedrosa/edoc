Sobre o Projeto

O E-doc é uma solução para microempresas que enfrentam dificuldades em unificar documentos, como notas fiscais e ordens de serviço. O objetivo é criar um programa que permita consolidar esses documentos em um formato eletrônico único e acessível.

Informações do Projeto

Nome do Projeto:

E-doc

Público-Alvo:

Microempresas

Problema:

Dificuldade de unificar documentos como notas fiscais e ordens de serviço.

Solução:

Criar um programa que unifique os documentos em um chamado documento eletrônico.

Tecnologias:

C#, Windows

Monetização:

Suporte anual

Requisitos Não Funcionais:

  • Usabilidade
  • Segurança
  • Confiabilidade

Requisitos Funcionais:

  • Cadastro de documentos
  • Geração de PDF

UML - Caso de Uso:

O caso de uso do sistema envolve o cadastro de documentos e sua conversão em formato eletrônico, possibilitando a geração de PDFs para envio e armazenamento.

Requisitos de Classes:

  • Banco de dados
  • Funções
